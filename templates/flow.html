Flow

Pre-processing:

cloud_connection.py

Downloading raw attendance csv files that are generated after class meetings and saved on Amazon remote machine, using SFTP.

Data-processing:

attendance.py

Runnig class attendance algorithm which was writen by Yona Bloy(Original file) on the raw attendance files. Summarized attendance.csv output file is generated.

Data-storing:

attendance_2_mysql.py

The output file is stored in Mysql database.

Client presentation:

flask_app.py

The data from the database is presented to the user in form of a web service over port 5000.

Application dockerization:

Dockerfile    docker-compose.yml

My docker image is based on ubuntu:22.04 image. I added pyhton3, mysql-client and the application requirment modules that i used to the image.
My flask web service runs as two containers with shared network. One runs Mysql server and the other one runs flask web application.
